## Installation

ViewDesign is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod "ViewDesign"
# Latest on develop
pod "ViewDesign", :git => 'https://github.com/gowthamravi/Viewdesign.git'

```
